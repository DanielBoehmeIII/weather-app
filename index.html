Whether
This website, Whether, was made as part of The Odin Project, a free web-development tutorial. I used the Visual Crossing API with AJAX to fetch data from local forecasters and utilize it through response.json. I don't expect this to become a prevalent source for weather data, however, if it gains enough stars on GitHub, I would be happy to refactor some things and incorporate new features. I hope you enjoy! Please check out more of my projects on my GitHub and follow me on Instagram and LinkedIn, the links to which you will find below.
